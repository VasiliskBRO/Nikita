Неплохо дать имя устройству с помощью добавления или изменения имени в конфиге nano/etc/hostname
.
(Имя)
.
Для взаимодействия с другими устройствами, нужно либо дать либо получить адрес и шлюз для этого необходимо отредактировать конфиг nano/etc/network/ineterfaces
 .
auto (интерфейс)
iface (интерфейс) inet (static/dhcp, выдать/получить)
address (адрес)/(маска)
gateway (шлюз)
.
Далее необходимо перезапустить конфиг командой
systemctl restart networking.service
Также для доступности доменных имен необходимо указать DNS сервер в конфиге 
nano /etc/resolv.conf
.
nameserver (адрес DNS сервера)
.
Далее проверить сколько адресов на интерфейсы командой 
Ip a
Если больше одного перезапустить
Reboot
Также для добавления новых репозиториев, чтобы можно было установить командой, без добавления ссылки на сайт скачивания используется команда
apt update
А для обновления из этих репозиториев используется команда
apt upgrade
Также можно указать часовой пояс и регион для получения мошенниками большей информации о вас, а также для лучшей пунктуальности, для серверов командой 
dpkg-reconfigure tzdata
А для клиентов через настройки по пути 
settings-DateTime-TimeZone-(часовой пояс)
