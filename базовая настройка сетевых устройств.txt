Для наименования устройств используется команда
hostname (имя устройства)
Также стоить включить интерфейс
no shutdown 
Еще для выставления времени используется команда
clock timezone (название можно число UTC) (число смещение в UTC) (число смещение в минутах)
Для роутеров используется ip адресация ее можно настроить командами
int (номер интерфейса)
ip add (адрес) (маска)
Если необходимо работать с волнами, то следует использовать субинтерфейсы
int (номер интерфейса к свитчу, на котором есть этот волан).(№ волана свитча)
encapsulation dot1Q (№ волана свитча)
ip add (адрес)(маска)
no shutdown
Для свитчей используется маркирование пакетов с помощью VLAN, для сегментации сетей.
Для этого создаться воланы, при необходимости можно задать имя и Ip адрес
Vlan (номер волана)
Name (имя волана)
ip add (адрес) (маска)
Ex
Если выдать адрес на волан, то необходимо прописать для отключения конфликтующей службы
no cdp run
Далее необходимо настроить интерфейсы, а именно определить сколько меток будет нанесено на пакет этого интерфейса
Для интерфейсов к сетевому оборудования обычно используют маркирование всеми метками
int (номер интерфейса)
switchport trunk encapsulation dot1q
switchport mode trunk
no sh
Для интерфейсов в сторону рабочих станций или серверов обычно используют маркирование одним воланом
switchport mode access
switchport access vlan (№ vlan)
interface vlan (№ vlan)
ip add (адрес) (маска)
no shutdown
